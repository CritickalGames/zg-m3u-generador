# 🎮 ROM Organizer & M3U Generator

Herramienta en Python con interfaz gráfica (PyQt6) que escanea una carpeta de ROMs, agrupa los juegos por franquicia usando búsqueda fuzzy (`rapidfuzz`), muestra el resultado en un árbol editable y genera archivos `.m3u` organizados.

## 📦 Instalación

pip install -r requirements.txt
